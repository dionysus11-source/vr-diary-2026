# 환율 추적 프로그램

원화 → 달러 환전 기록을 관리하고, 달러 사용을 추적하며 재환전 시 이익을 계산하는 콘솔 프로그램입니다.

## 기능

- ✅ 환율 기록 추가 (날짜, 환율, 금액)
- ✅ 달러 사용 기록 추가 (날짜, 금액, 용도, 메모)
- 📋 전체 기록 조회 ([IN] 환전 / [OUT] 사용 구분, 색상 표시)
- 📊 통계 정보 (총 환전, 총 사용, 현재 보유, 평균 환율)
- 💹 재환전 이익 계산 (현재 보유 달러로만 계산, 색상 표시)
- 🔒 잔액 검증 (보유 달러보다 큰 금액 사용 불가)
- 🗑️ 데이터 초기화 (이중 확인 절차)

## 설치

Python 3.7 이상이 필요합니다.

```bash
# 별도의 의존성 없이 순수 Python으로 작성됨
python scripts/exchange_rate_tracker/exchange_rate_tracker.py
```

## 사용 방법

프로그램을 실행하면 메뉴가 표시됩니다:

```
[APP] 환율 추적 프로그램
======================================================================
1. 환율 기록 추가 (원화 → 달러)
2. 달러 사용 기록 추가 (달러 사용)
3. 전체 기록 보기
4. 통계 보기
5. 재환전 이익 계산
6. 데이터 초기화
7. 종료
======================================================================
```

### 1. 환율 기록 추가

원화로 달러를 환전한 기록을 추가합니다.

```
[ADD] 환율 기록 추가
----------------------------------------------------------------------
날짜 (YYYY-MM-DD, 기본값: 2026-03-13): 2026-03-01
환율 (원/달러): 1300
환전 금액 (달러): 1000

[OK] 기록이 추가되었습니다: 2026-03-01 - 1300.0원/$, $1000.00
```

### 2. 달러 사용 기록 추가

보유 달러를 사용한 기록을 추가합니다 (주식, ETF 등).

```
[USAGE] 달러 사용 기록 추가
----------------------------------------------------------------------
[INFO] 현재 보유 달러: $1500.00
날짜 (YYYY-MM-DD, 기본값: 2026-03-13): 2026-03-10
사용 금액 (달러): 300

[USAGE] 자주 사용하는 용도:
  1. 주식 매수
  2. ETF 매수
  3. 배당금 재투자
  4. 기타
용도를 선택하세요 (1-4, 기본값: 4): 1
메모 (선택사항): Tesla 10주

[OK] 사용 기록이 추가되었습니다: 2026-03-10 - $300.00 (주식 매수)
```

### 3. 전체 기록 보기

모든 환전 기록과 사용 기록을 날짜순으로 표시합니다.

```
[LIST] 전체 기록 보기
================================================================================
날짜           타입       환율(원/$)      금액($)        용도/메모
--------------------------------------------------------------------------------
2026-03-01   [IN]          1,300    1,000.00  환전
2026-03-05   [IN]          1,280      500.00  환전
2026-03-10   [OUT]             -      300.00  TSLA 주식 매수
================================================================================
```

### 4. 통계 보기

현재까지의 환전/사용 통계를 표시합니다.

```
[STATS] 환전 통계
======================================================================
환전 횟수:       2회
사용 횟수:       1회
----------------------------------------------------------------------
총 환전 금액:   $1,500.00
총 사용 금액:   $300.00
현재 보유 달러: $1,200.00
----------------------------------------------------------------------
총 투자 원화:   ₩1,940,000
평균 환율:      ₩1,293.33/$ (변하지 않음)
======================================================================
```

**중요:** 평균 환율은 총 투자 원화 ÷ 총 환전 금액으로 계산되며, 달러 사용과 무관하게 변하지 않습니다.

### 5. 재환전 이익 계산

목표 환율을 입력하면 **현재 보유 달러**로 재환전 시 이익/손실을 계산합니다.

```
[PROFIT] 현재 보유: $1,200.00, 평균 환율: ₩1,293/$
목표 환율 (원/달러): 1400

[PROFIT] 재환전 이익 계산
======================================================================
현재 보유 달러:     $1,200.00
총 투자 원화:       ₩1,940,000
평균 환율:          ₩1,293.33/$ (변하지 않음)
목표 환율:          ₩1,400/$
----------------------------------------------------------------------
재환전 후 원화:      ₩1,680,000
예상 [DOWN] 손실:     ₩260,000 (-13.40%)
======================================================================
```

## 데이터 저장

모든 데이터는 `exchange_records.csv` 파일에 저장됩니다.

### CSV 파일 형식

```csv
type,date,rate,amount_usd,amount_krw,purpose,note
exchange,2026-03-01,1300.0,1000.0,1300000.0,,
exchange,2026-03-05,1280.0,500.0,640000.0,,
usage,2026-03-10,0,300.0,0,TSLA 주식 매수,Tesla 10주
```

**필드 설명:**
- `type`: 기록 유형 (`exchange` = 환전, `usage` = 사용)
- `date`: 날짜
- `rate`: 환율 (원/달러, usage인 경우 0)
- `amount_usd`: 달러 금액
- `amount_krw`: 원화 금액 (usage인 경우 0)
- `purpose`: 사용 용도 (exchange인 경우 공백)
- `note`: 메모 (선택사항)

## 사용 예시

### 시나리오: 분할 매수 → 주식 투자 → 재환전

1. **3월 1일**: 1,300원/$ × 1,000달러 환전
2. **3월 5일**: 1,280원/$ × 500달러 환전
3. **3월 10일**: 300달러로 TSLA 주식 매수
4. **3월 15일**: 환율이 1,400원/$로 상승

### 결과 분석

```
총 환전: 1,500달러
총 사용: 300달러
현재 보유: 1,200달러  ← 재환전 계산 시 사용
평균 환율: 1,293.33원/$ (변하지 않음)

재환전 계산 (1,400원/$):
  1,200달러 × 1,400원 = 1,680,000원
  총 투자: 1,940,000원
  예상 손실: -260,000원 (-13.40%)
```

### 핵심 원리

**평균 환율 계산:**
```
평균 환율 = 총 투자 원화 ÷ 총 환전 금액
        = 1,940,000원 ÷ 1,500달러
        = 1,293.33원/$
```

달러를 사용해도 평균 환율은 변하지 않습니다. 이는 "내가 달러를 산 평균 가격"을 유지하면서, 현재 보유 달러만으로 재환전 수익을 계산하기 위함입니다.

### 6. 데이터 초기화

모든 기록을 안전하게 삭제할 수 있습니다 (이중 확인 절차).

```
[RESET] 데이터 초기화
======================================================================
총 환전 횟수:  2회
사용 횟수:      1회
현재 보유:     $600.00
총 투자 원화:   ₩870,000
----------------------------------------------------------------------
[WARNING] 모든 데이터가 영구적으로 삭제됩니다!
[WARNING] 이 작업은 되돌릴 수 없습니다!
======================================================================

정말로 모든 데이터를 삭제하시겠습니까? (yes/no): yes

[WARNING] 마지막 확인입니다!
[WARNING] 정말로 모든 데이터를 삭제하시겠습니까?
======================================================================

삭제를 진행하려면 'DELETE'를 입력하세요: DELETE

[SUCCESS] 모든 데이터가 삭제되었습니다.
======================================================================
```

**안전 장치:**
1. 첫 번째 확인: `yes` 또는 `y` 입력
2. 두 번째 확인: `DELETE` 정확히 입력
3. 언제든지 취소 가능
4. 빈 데이터인 경우 경고 메시지 표시


## 주의사항

- 환율과 금액은 양수여야 합니다
- 날짜는 YYYY-MM-DD 형식으로 입력하세요 (기본값: 오늘)
- 보유 달러보다 큰 금액을 사용할 수 없습니다 (잔액 부족 오류)
- CSV 파일을 직접 수정하지 마세요
- 정확한 계산을 위해 모든 환전 기록을 입력하세요
- 평균 환율은 달러 사용과 무관하게 변하지 않습니다

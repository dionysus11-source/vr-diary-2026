# Change: 무한 매수 기록 시각화 및 분석 대시보드 추가

## 왜 (Why)
현재 무한 매수 기록을 시각적으로 확인할 수 있는 방법이 없습니다. 회차별 수익률 변화와 누적 수익금을 한눈에 파악할 수 있는 대시보드가 필요합니다.

## 무엇을 변경하는가 (What Changes)
- **BREAKING** 없음 (신규 기능 추가)
- 무한 매수 기록 데이터 수집 및 저장 기능 추가
- 회차별 수익률 그래프 시각화 추가
- 누적 수익금 추이 차트 추가
- 회차 요약 정보 표시 (최대 손실률, 최대 수익률, 평균 수익률 등)
- 데이터 필터링 및 기간 선택 기능 추가

## 영향 (Impact)
- 영향받는 사양: 새로운 `infinite-buying-analytics` capability 생성
- 영향받는 코드:
  - 새로운 데이터 수집 모듈 (`src/data/collector.ts`)
  - 새로운 시각화 컴포넌트 (`src/components/charts/`)
  - 새로운 분석 로직 (`src/services/analyzer.ts`)

## 추가로 제안하는 유용한 정보

### 기본 필수 정보
1. **회차별 수익률 그래프** - 각 회차의 최종 수익률을 선/막대 그래프로 표시
2. **누적 수익금 추이** - 전체 기간 동안의 자산 변화를 시각화
3. **회차 요약 통계** - 최대/최소/평균 수익률, 승률, 최대 손실폭(MDD)

### 고급 분석 정보 (권장)
4. **구간별 분석** - N회차 이동평균 수익률, 특정 구간 성과 비교
5. **분산 분석** - 수익률 표준편차로 변동성 지표 제공
6. **손실 복구 기간** - 최대 손실 후 원금 회복까지 걸린 회차 수
7. **연환산 수익률** - 회차 기간을 고려한 연간 환산 수익률

### 위험 관리 정보
8. **MDD(Maximum Drawdown) 차트** - 최대 손실폭을 시각적으로 표시
9. **회당 리스크** - 평균 손실액, 평균 수익액 비교
10. **손절/익절 비율** - 손절로 끝난 회차 vs 익절로 끝난 회차 비율

### 예측 및 시뮬레이션
11. **몬테카를로 시뮬레이션** - 과거 데이터 기반 미래 성과 시뮬레이션
12. **목표 달성 확률** - 특정 수익 목표 달성까지 필요한 회차 수 예측

# Change: 밸류 리밸런싱 웹 페이지 추가

## Why
현재 무한 매수법 사이트에 밸류 리밸런싱 전략을 추가하여 사용자가 두 가지 투자 전략을 비교하고 선택할 수 있게 해야 한다. 밸류 리밸런싱 CLI 프로그램이 이미 존재하므로 이를 웹 인터페이스로 통합하여 사용자가 더 편리하게 사용할 수 있게 한다.

## What Changes
- **왼쪽 사이드바 네비게이션 추가**: 왼쪽에서 무한 매수법과 밸류 리밸런싱 선택
- **밸류 리밸런싱 전용 페이지 추가**: /value-rebalancing 라우트로 새로운 페이지 생성
- **V값 계산 기능**: 라오어 밸류 리밸런싱 공식 (v2 = v1 + pool/10 + (e-v1)/6.32) 구현
- **리밸런싱 신호 시스템**: V값 변화 ±15% 기준으로 BUY/SELL/HOLD 신호 제공
- **벤치마크 비교**: 초기 전액 TQQQ 투자 시나리오와 실제 VR 전략 성과 비교
- **차수 관리**: 2주 단위 차수별 매수/매도 기록 추적
- **시계열 차트**: V값 변화, 평가금액, 벤치마크 비교 차트 제공
- **데이터 영구 저장**: 리밸런싱 기록, 초기 포트폴리오, 차수 정보 저장

## Impact
- **Affected specs**:
  - `value-rebalancing` (new capability)
- **Affected code**:
  - `app/layout.tsx`: 왼쪽 사이드바 레이아웃 추가 (기존 페이지 감싸기)
  - `app/value-rebalancing/page.tsx`: 밸류 리밸런싱 전용 페이지 (새로 생성)
  - `components/ui/`: 사이드바 및 밸류 리밸런싱 컴포넌트 추가
  - `lib/calculations/`: V값 계산 로직 추가
  - `lib/data/`: 리밸런싱 데이터 관리 추가
  - `types/index.ts`: 밸류 리밸런싱 타입 추가

## Non-Goals
- **기존 무한 매수법 페이지 변경 없음**: `app/page.tsx`는 전혀 수정하지 않음
- **기존 컴포넌트 재사용 없음**: 무한 매수법 컴포넌트와 완전히 분리
- **실시간 API 연동 없음**: CLI처럼 수동 가격 입력 또는 선택적 API 연동
- **복잡한 포트폴리오 관리 없음**: 단일 TQQQ 종목에 집중

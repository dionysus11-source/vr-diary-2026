## MODIFIED Requirements

### Requirement: Console User Interface
시스템 SHALL 사용자 친화적인 콘솔 인터페이스를 제공해야 한다.

#### Scenario: 메인 메뉴 표시
- **WHEN** 프로그램이 시작된다
- **THEN** 시스템은 다음 옵션들을 표시한다:
  - 1. 환율 기록 추가
  - 2. 달러 사용 기록 추가
  - 3. 전체 기록 보기
  - 4. 통계 보기
  - 5. 재환전 이익 계산
  - 6. 데이터 초기화
  - 7. 종료

## ADDED Requirements

### Requirement: Data Reset Functionality
시스템 SHALL 사용자가 모든 기록을 안전하게 초기화할 수 있어야 한다.

#### Scenario: 데이터 초기화 성공
- **WHEN** 사용자가 데이터 초기화를 선택한다
- **THEN** 시스템은 첫 번째 경고 메시지를 표시한다
- **AND** "모든 데이터가 영구적으로 삭제됩니다" 경고를 포함한다
- **AND** 현재 보유 달러와 총 기록 수를 표시한다
- **AND** 사용자에게 계속 진행할 것인지 묻는다
- **WHEN** 사용자가 확인한다 (예: 'yes' 입력)
- **THEN** 시스템은 두 번째 확인을 요청한다
- **AND** "정말로 모든 데이터를 삭제하시겠습니까?" 메시지를 표시한다
- **WHEN** 사용자가 다시 확인한다
- **THEN** 시스템은 CSV 파일을 삭제한다
- **AND** 메모리 상태를 초기화한다
- **AND** 성공 메시지를 표시한다

#### Scenario: 데이터 초기화 - 첫 번째 확인에서 취소
- **WHEN** 사용자가 데이터 초기화를 선택한다
- **THEN** 시스템은 첫 번째 경고 메시지를 표시한다
- **WHEN** 사용자가 취소한다 (예: 'no' 입력)
- **THEN** 시스템은 "초기화가 취소되었습니다" 메시지를 표시한다
- **AND** 메인 메뉴로 돌아간다
- **AND** 데이터는 유지된다

#### Scenario: 데이터 초기화 - 두 번째 확인에서 취소
- **WHEN** 사용자가 데이터 초기화를 선택한다
- **AND** 첫 번째 확인을 완료한다
- **AND** 두 번째 확인을 요청받는다
- **WHEN** 사용자가 취소한다
- **THEN** 시스템은 "초기화가 취소되었습니다" 메시지를 표시한다
- **AND** 메인 메뉴로 돌아간다
- **AND** 데이터는 유지된다

#### Scenario: 데이터 초기화 - 빈 데이터베이스
- **WHEN** 사용자가 데이터 초기화를 선택한다
- **AND** 저장된 기록이 없다
- **THEN** 시스템은 "삭제할 데이터가 없습니다" 메시지를 표시한다
- **AND** 메인 메뉴로 돌아간다

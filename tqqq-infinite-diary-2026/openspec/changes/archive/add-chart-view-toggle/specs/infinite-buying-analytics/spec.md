## MODIFIED Requirements

### Requirement: 회차 상세 모달
시스템 SHALL 카드 클릭 시 회차 상세를 모달로 표시해야 한다(MUST).

#### Scenario: 모달 내용 - 시계열 수익률 차트
- **WHEN** 회차 상세 모달이 열린다
- **THEN** 해당 회차의 시간대별 데이터를 차트로 표시한다
- **AND** X축은 시간(일자)을 나타낸다
- **AND** 차트 상단에 뷰 전환 토글을 제공한다: "수익률" / "가격"
- **AND** 기본 뷰는 "수익률"로 설정한다
- **AND** 사용자가 토글을 클릭하면 해당 뷰로 전환된다
- **AND** 진행중 회차는 최신 종가로 차트를 자동 업데이트한다
- **AND** 완료된 회차는 최종 매도일까지의 추이를 보여준다
- **AND** API 로딩 중임을 표시한다
- **AND** 날짜는 한국시간 기준으로 표시된다

#### Scenario: 차트 뷰 - 수익률 모드
- **WHEN** 사용자가 "수익률" 뷰를 선택한다
- **THEN** Y축(오른쪽)은 수익률(%)을 나타낸다
- **AND** 수익률 영역 차트(Area)를 표시한다
- **AND** 수익 구간은 초록색, 손실 구간은 빨간색으로 표시한다
- **AND** 0% 수익률 기준선을 명확히 표시한다
- **AND** 종가와 평단가 선은 표시하지 않는다
- **AND** 하단 범례에 "수익률"을 표시한다

#### Scenario: 차트 뷰 - 가격 모드
- **WHEN** 사용자가 "가격" 뷰를 선택한다
- **THEN** Y축(왼쪽)은 가격($)을 나타낸다
- **AND** 종가 선 그래프(파란색)를 표시한다
- **AND** 평균 매수가 기준선(주황색)을 수평 점선으로 명확히 표시한다
- **AND** 각 매수 시점에 "매수" 마커를 표시한다
- **AND** 각 매도 시점에 "매도" 마커를 표시한다
- **AND** 수익률 영역 차트와 Y축은 표시하지 않는다
- **AND** 하단 범례에 "종가", "평단가"를 표시한다

#### Scenario: 시계열 차트의 데이터 포인트 상세 보기 - 수익률 뷰
- **WHEN** 사용자가 "수익률" 뷰에서 차트의 데이터 포인트에 마우스를 올린다 (hover)
- **THEN** 해당 시점의 상세 정보를 tooltip으로 표시한다
- **AND** tooltip에는 다음 정보가 포함된다:
  - 일자 (한국시간 기준)
  - 수익률 (예: +7.11%)
  - 수익금 (예: +$3.22)
  - 해당 시점의 사건 (매수/매도/없음)
- **AND** 날짜는 한국시간 기준으로 표시된다

#### Scenario: 시계열 차트의 데이터 포인트 상세 보기 - 가격 뷰
- **WHEN** 사용자가 "가격" 뷰에서 차트의 데이터 포인트에 마우스를 올린다 (hover)
- **THEN** 해당 시점의 상세 정보를 tooltip으로 표시한다
- **AND** tooltip에는 다음 정보가 포함된다:
  - 일자 (한국시간 기준)
  - 종가 (예: $48.52)
  - 평균 매수가 (예: $45.30)
  - 수익률 (예: +7.11%)
  - 해당 시점의 사건 (매수/매도/없음)
- **AND** 날짜는 한국시간 기준으로 표시된다

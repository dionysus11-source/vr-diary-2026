## ADDED Requirements
### Requirement: 현재 포트폴리오 수동 동기화
시스템 SHALL 사용자가 실제 계좌 상황과 다를 경우 주식수 및 현금(Pool)을 직접 수정하여 현재 포트폴리오 상태를 동기화할 수 있게 해야 한다(MUST).

#### Scenario: 현재 포트폴리오 정보 수정 시작
- **WHEN** 사용자가 현재 포트폴리오 상태 카드에서 "수정" 버튼을 클릭한다
- **THEN** 주식수와 현금(Pool)을 입력받을 수 있는 폼 모달을 표시한다
- **AND** 각 입력 필드는 현재(가장 최신) 포트폴리오 값으로 초기화되어 있어야 한다

#### Scenario: 포트폴리오 수정 완료
- **WHEN** 사용자가 수정된 주식수 및 현금을 입력하고 "저장" 버튼을 클릭한다
- **THEN** 서버 API를 호출하여 최신 기록(VRRecord)의 주식수와 현금을 변경된 값으로 업데이트한다
- **AND** 현재 기록의 eValue 및 totalValue 등 종속 데이터도 새 값에 맞게 재계산되어야 한다
- **AND** 업데이트 완료 시 화면에 반영된 최신 포트폴리오 상태가 표시된다

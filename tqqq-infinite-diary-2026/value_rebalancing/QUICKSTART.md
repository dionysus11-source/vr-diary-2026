```
1. 초기 설정 & 첫 V값 계산 (벤치마크 포함)
```

다음 정보를 입력하세요:
- **초기 날짜**: YYYY-MM-DD (엔터 시 오늘)
- **초기 현금(Pool)**: 보유 현금
- **초기 주식수**: 보유 TQQQ 주식수
- **평단가**: 평균 매수 단가

자동으로 계산됩니다:
- ✅ 첫 V값
- ✅ 벤치마크 (전액 TQQQ 투자 가정)

### 2단계: 주말 종가 기준 V값 계산
```
2. 주말 종가 기준 V값 계산 & 기록
```

입력 정보:
- **현금(Pool)**: 현재 보유 현금
- **주식수**: 현재 보유 주식수
- **날짜**: 계산할 날짜
- **가격**: 자동 조회 또는 직접 입력

자동으로 계산됩니다:
- ✅ V2 (새 V값)
- ✅ 리밸런싱 신호 (BUY/SELL/HOLD)
- ✅ 벤치마크 업데이트
- ✅ VR vs 벤치마크 비교

### 3단계: VR vs 벤치마크 조회
```
3. 날짜별 VR vs 벤치마크 조회
```

확인할 수 있는 정보:
- 📊 날짜별 V값 추이
- 📊 날짜별 벤치마크 가치
- 📊 VR vs 벤치마크 차이
- 📊 누적 수익률 비교
- 📊 VR 초과 수익률

### 4단계: 예약 주문 시뮬레이션
```
4. 예약 주문 시뮬레이션 (V값 기준)
```

확인할 수 있는 정보:
- 📊 V값 기준 주문당 예약 주문 가격
- 📊 BUY 시나리오: 매수 밴드(-15%)에서 N주씩 매수 시 Pool 변화
- 📊 SELL 시나리오: 매도 밴드(+15%)에서 N주씩 매도 시 Pool 변화

입력 항목:
- V값 (자동으로 ±15% 밴드 계산)
- 보유 주식수
- 현재 Pool
- Pool 한도
- 주문 단위 (예: 1주, 10주, 50주)

계산 방식:
- 매수 밴드 = V값 × 0.85
- 매도 밴드 = V값 × 1.15
- 1주당 예약가 = 밴드 V값 ÷ 현재 주식수
- N주당 금액 = 1주당 예약가 × N
- 매수: Pool - N주당 금액, 주식수 증가 (Pool 한도 내에서 반복)
- 매도: Pool + N주당 금액, 주식수 감소 (보유 주식의 30% 한도)

## 3. 핵심 개념 이해

### VR (Value Rebalancing)
- **V값**: `V2 = v1 + (pool/10) + ((E-v1)/(2×√10))`
- **Pool**: 현금
- **E (평가금액)**: 종가 × 주식수
- **신호**: V2가 V1 대비 ±15% 벗어나면 매수/매도

### 벤치마크 (Benchmark)
- **개념**: 초기 총 투자 금액을 전부 TQQQ에 투자했다고 가정
- **벤치마크 주식수**: 총 투자액 / 초기 종가
- **벤치마크 가치**: 벤치마크 주식수 × 현재 종가
- **비교**: VR이 벤치마크보다 얼마나 잘했는지 확인

### 예시
```
초기 설정:
- 현금: $100,000
- 주식수: 100주
- 평단가: $40.50
- 총 투자: $104,050

벤치마크:
- 2,568.52주를 $40.50에 매수 (전액 TQQQ 투자)

1주 후:
- TQQQ 종가: $45.32 (+11.93%)
- 벤치마크 가치: $116,407 (+11.93%)
- VR V값: $54,532 (-47.62%)
- 결론: 이번 경우 벤치마크가 우수
```

# 📊 라오어 밸류 리밸런싱 vs 벤치마크 CLI

TQQQ 레버리지 ETF 투자에서 **밸류 리밸런싱(VR) 전략**과 **단순 TQQQ 투자(벤치마크)**의 성과를 비교하는 CLI 프로그램입니다.

## 🎯 주요 기능

1. **초기 설정 & 첫 V값 계산**: 벤치마크와 함께 첫 V값 자동 계산
2. **주말 종가 기준 V값 계산**: 직접 입력 또는 API 자동 조회
3. **VR vs 벤치마크 비교**: 날짜별 성과 비교 및 초과 수익률 확인
4. **5주 분할 매수 시뮬레이션**: 다양한 가격 시나리오에서의 매수/매도 시점 예측
5. **가격 데이터 관리**: 야후 파이낸스 API 연동 및 CSV 캐싱
6. **CSV 데이터 영구 저장**: 모든 기록을 CSV 파일로 저장

## 📐 밸류 리밸런싱 공식

```
V2 = v1 + (pool/10) + ((E-v1)/(2×√10))
```

- **E (평가금액)**: 종가 × 주식수
- **v1**: 이전 V값 (첫 계산 시 = 평가금액)
- **pool**: 현금
- **G**: 10 (고정)

## 🏆 벤치마크 개념

**벤치마크**: 초기 총 투자 금액(initial_cash + 주식가치)을 전부 TQQQ에 투자했을 때의 가치

- **벤치마크 주식수** = 총 투자 금액 / 초기 종가
- **벤치마크 가치** = 벤치마크 주식수 × 현재 종가
- **VR 초과 수익률** = VR 수익률 - 벤치마크 수익률

## 🔧 기술 스택

- **Python 3.8+**

## ⚠️ 주의사항

1. **데이터 백업**: 모든 데이터는 로컬 CSV 파일에 저장됩니다. 정기적으로 백업하세요.
2. **API 제한**: 야후 파이낸스 API는 무료이지만 과도한 요청은 제한될 수 있습니다.
3. **오프라인 모드**: 인터넷 연결이 없을 때는 캐시된 가격 데이터를 사용합니다.
4. **투자 책임**: 이 도구는 정보 제공만 목적이며, 투자 결정의 책임은 사용자에게 있습니다.

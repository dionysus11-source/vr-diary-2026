# 무한 매수 다이어리 2026

TQQQ/SOXL 무한 매수 회차별 수익률 추적 및 분석 웹앱

## 주요 기능

### 회차 관리
- 새 회차 시작 (TQQQ/SOXL)
- 분할 매수/매도 기록
- 진행중인 회차의 현재 가격 업데이트
- 회차 상태 관리 (진행중/완료)

### 수익률 분석
- 실현/미실현 수익률 자동 계산
- 평균 매수 단가 추적
- 총 수익금 및 수익률 통계
- 승률, 표준편차, MDD, 손익비 등 고급 통계

### 시각화
- 누적 수익금 추이 차트
- 최대 손실폭 (MDD) 차트
- 시계열 수익률 분석

### 데이터 관리
- 백업 (JSON 파일로 내보내기)
- 복원 (백업 파일에서 가져오기)
- CSV 내보내기 지원

### UI/UX
- 다크/라이트 모드 지원
- 간결/상세 보기 모드
- 반응형 디자인 (모바일 지원)
- 한국시간 기반 날짜 처리

## 기술 스택

- **프레임워크**: Next.js 15
- **UI 라이브러리**: React 19
- **언어**: TypeScript
- **스타일링**: Tailwind CSS
- **차트**: Recharts
- **데이터 저장**: localStorage

## 시작하기

### 설치

```bash
npm install
```

### 개발 모드 실행

```bash
npm run dev
```

앱이 http://localhost:5000에서 시작됩니다.

### 프로덕션 빌드

```bash
npm run build
npm run start
```

프로덕션 서버가 http://localhost:5000에서 시작됩니다.

### 💻 새 PC에서 세팅하기 (이전 데이터 연동법)

다른 컴퓨터 또는 포맷된 PC에서 이 프로젝트를 새로 세팅하려면 다음 순서를 따라주세요.

1. **저장소 클론 및 패키지 설치**
   ```bash
   git clone [저장소 주소]
   cd tqqq-infinite-diary-2026
   npm install
   ```

2. **데이터베이스 초기화 (필수)**
   아직 내부 DB가 설정되어 있지 않으므로 Prisma 클라이언트를 생성하고 DB 스키마를 동기화해야 합니다.
   ```bash
   npx prisma generate
   npx prisma db push
   ```

3. **앱 실행**
   ```bash
   npm run build
   npm run start
   ```

4. **전체 데이터 복원 (동기화)**
   - 앱이 처음 실행되면 빈 데이터베이스 화면이 보입니다. (개인 ID 로그인 방식이 아니므로, 브라우저가 새 ID를 할당합니다)
   - 웹 접속 후 **상단의 "백업/복원" 버튼**을 누릅니다.
   - 기존 PC에서 추출했던 **최신 백업 파일(`.json`)을 업로드**합니다.
   - ⚠️ 복원이 성공하면 새로운 PC가 과거 PC의 `고유 ID`를 흡수하여 **영구적으로 두 PC가 하나의 데이터베이스를 공유**하게 됩니다!


## 데이터 백업/복원

### 백업 (내보내기)
1. 상단의 "백업/복원" 버튼 클릭
2. "백업 파일 다운로드" 버튼 클릭
3. JSON 파일이 다운로드됩니다

### 복원 (가져오기)
1. 상단의 "백업/복원" 버튼 클릭
2. 백업 JSON 파일 선택
3. "복원 실행" 버튼 클릭
4. 확인 대화상자에서 확인

⚠️ **주의**: 복원 시 현재 데이터가 모두 백업 파일로 대체됩니다.

## 계산 로직

### 수익률 계산
- **실현 수익률**: (총 매도액 - 총 매수액) / 총 매수액 × 100
- **미실현 수익률**: (현재 가치 - 총 매수액) / 총 매수액 × 100
- **총 수익률**: 실현 + 미실현 수익률

### 평균 매수 단가
- 총 매수액 / 총 매수 수량

### MDD (최대 손실폭)
- (고점 - 저점) / 고점 × 100
